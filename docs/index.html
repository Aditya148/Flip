üöÄ Quick Start

Installation

Install Flip SDK using pip:

pip install flip-rag

Basic Usage

Get started with Flip in just a few lines of code:

from flip import Flip

# Initialize with a directory
flip = Flip(directory="./docs")

# Query your documents
response = flip.query("What is the main topic?")

# Access the results
print(response.answer)
print(response.citations)

üìö Supported LLM Providers

Choose from multiple LLM providers with a simple parameter change:

# Use OpenAI (default)
flip = Flip(directory="./docs", llm_provider="openai")

# Environment variable: OPENAI_API_KEY
from flip import Flip, FlipConfig

config = FlipConfig(
    llm_provider="azure-openai",
    llm_model="gpt-4",  # Your deployment name
    embedding_provider="azure-openai",
    embedding_model="text-embedding-ada-002",
    azure_openai_api_key="your-key",
    azure_openai_endpoint="https://your-resource.openai.azure.com/"
)

flip = Flip(directory="./docs", config=config)

flip = Flip(directory="./docs", llm_provider="anthropic")

# Environment variable: ANTHROPIC_API_KEY

# Environment variable: GOOGLE_API_KEY

flip = Flip(directory="./docs", llm_provider="huggingface")

# Environment variable: HUGGINGFACE_API_KEY
# Use local Ollama
flip = Flip(directory="./docs", llm_provider="ollama")

# No API key needed - runs locally!

Flip Class

The main SDK class for RAG operations.

flip = Flip(
    directory: Optional[str] = None,
    config: Optional[FlipConfig] = None,
    llm_provider: Optional[str] = None,
    llm_model: Optional[str] = None,
    **kwargs
)

Methods

query(question: str) ‚Üí FlipResponse

Query your documents and get an AI-generated answer.

index_directory(directory: str)

Index all documents in a directory.

add_documents(file_paths: List[str])

Add specific files to the index.

refresh_index()

Incrementally update index with changed documents.

evaluate(query, relevant_ids, k=5)

Evaluate RAG performance with metrics.

get_monitoring_stats()

Get performance statistics.

clear()

Clear all indexed documents.

get_stats()

Get indexing statistics.

FlipResponse

Response object returned by query operations.

@dataclass
class FlipResponse:
    answer: str              # Generated answer
    citations: List[Dict]    # Source citations
    context_chunks: List[str] # Retrieved chunks
    metadata: Dict[str, Any] # Additional metadata

FlipConfig

Configuration object for customizing Flip behavior.

from flip import FlipConfig

config = FlipConfig(
    # LLM Configuration
    llm_provider="openai",
    llm_model="gpt-4-turbo-preview",
    llm_temperature=0.7,
    llm_max_tokens=1024,
    
    # Embedding Configuration
    embedding_provider="openai",
    embedding_model="text-embedding-3-small",
    
    # Vector Store
    vector_store="chroma",
    vector_store_path="./flip_data",
    
    # Chunking
    chunking_strategy="semantic",  # token, sentence, semantic, recursive
    chunk_size=512,
    chunk_overlap=50,
    
    # Retrieval
    retrieval_top_k=5,
    use_hybrid_search=True,
    use_reranking=True,
    
    # Cache
    enable_cache=True,
    cache_dir="./flip_cache"
)

üöÄ Advanced Features

üîÄ Hybrid Search

Combines vector similarity search with BM25 keyword search using reciprocal rank fusion for superior accuracy.

How It Works

  • Vector Search: Semantic similarity using embeddings
  • Keyword Search: BM25 algorithm for exact matches
  • Fusion: Reciprocal Rank Fusion combines both results
from flip import Flip, FlipConfig

config = FlipConfig(
    use_hybrid_search=True  # Enabled by default
)

flip = Flip(directory="./docs", config=config)

üìä Re-ranking

Uses cross-encoder models to re-rank retrieved documents for improved relevance.

config = FlipConfig(
    use_reranking=True,  # Enabled by default
    reranker_model="cross-encoder/ms-marco-MiniLM-L-6-v2"
)

flip = Flip(directory="./docs", config=config)

üíæ Intelligent Caching

LRU caching for embeddings and queries with disk persistence for faster repeated operations.

Cache Types

  • Embedding Cache: Caches document embeddings
  • Query Cache: Caches query results
config = FlipConfig(
    enable_cache=True,  # Enabled by default
    cache_dir="./flip_cache"
)

flip = Flip(directory="./docs", config=config)

# Manually save caches
flip.pipeline.save_caches()

# Clear caches
flip.pipeline.clear_caches()

üîÑ Incremental Updates

Efficiently track and update only changed documents instead of re-indexing everything.

flip = Flip(directory="./docs")

# Later, refresh with only changed documents
stats = flip.refresh_index()

print(f"Added: {stats['added']}")
print(f"Updated: {stats['updated']}")
print(f"Deleted: {stats['deleted']}")

Simple Q&A

from flip import Flip

flip = Flip(directory="./company_docs")
response = flip.query("What is our refund policy?")

Multiple Providers

from flip import Flip

providers = ["openai", "anthropic", "google"]

for provider in providers:
    flip = Flip(directory="./docs", llm_provider=provider)
    response = flip.query("Summarize the main points")
    print(f"{provider}: {response.answer[:100]}...")

Advanced Configuration

from flip import Flip, FlipConfig

config = FlipConfig(

    embedding_provider="openai",

Evaluation

from flip import Flip

flip = Flip(directory="./docs")

# Evaluate a query
result = flip.evaluate(
    query="What is AI?",
    relevant_doc_ids=["chunk_1", "chunk_2"],
    k=5
)

print(f"Precision: {result.retrieval_precision:.3f}")
print(f"Overall Score: {result.overall_score:.3f}")
